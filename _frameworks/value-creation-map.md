The Value Creation Map is the first executive artifact because it reconnects activity to customer, user, citizen, or institutional value, then links that value to cost, constraint, leverage, and tradeoff.

Use it before changing roles, buying tools, launching transformation programs, or redesigning meetings.

## First Questions

- Where does value come from?
- What turns effort into value?
- Where does value leak?
- What constrains us?
- What gives us leverage?

## First Move

Write a ten-page maximum map of the business as it actually works. Version it. Review it quarterly. Let decisions reference it.
